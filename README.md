Unofficial [Moves API](https://dev.moves-app.com) client written in Swift.

## Instalation

You can integrate `MovesAPI` with your project using [CocoaPods](https://cocoapods.org). Just add this line to your `Podfile`:

```ruby
pod 'MovesAPI'
```
